# Glossar

Die verbindliche Begriffsliste steht in [`index.md`](index.md). Neue Produktabkürzungen dürfen
erst nach bestätigter Langform, Zuständigkeit und Systemgrenze als verbindlich markiert werden.

Historische Bedeutung und heutiger Nachweisstatus werden getrennt in
[`term-status.md`](term-status.md) geführt. Quellenrekonstruktionen liegen unter `sources/` und
sind ohne aktuelle Repository-Prüfung kein Implementierungsnachweis.
