# Was ist der AAIA Module Manager?

## Kurz erklärt

Der **AAIA Module Manager** ist das offizielle Werkzeug für ETW-Entwickler. Er führt dich Schritt für Schritt durch den gesamten Prozess — von der ersten Idee bis zur Veröffentlichung auf dem Marketplace.

Stell ihn dir wie einen Assistenten vor, der neben dir sitzt und:
- prüft ob alles stimmt,
- erklärt was schiefgelaufen ist,
- automatisch repariert was möglich ist,
- und dich blockiert wenn etwas wirklich wichtiges fehlt.

## Was kann ich damit tun?

| Funktion | Beschreibung |
|----------|-------------|
| Projekt erstellen | Idee eingeben, Projektstruktur anlegen lassen |
| Validierung | Automatische Prüfung ob alles korrekt ist |
| Build | Quellcode kompilieren |
| Paketierung | `.aaiaext`-Datei erzeugen |
| Paketprüfung | Inhalt auf Sicherheit prüfen |
| Release | Alles für den Upload vorbereiten |
| Signatur | Paket digital signieren |
| Marketplace-Upload | Paket auf den Marketplace hochladen |
| KI-Assistent | Hilfsprompts für Claude, ChatGPT oder Gemini erzeugen |
| Connector | Angeschlossene KI-Agenten steuern |

## Wer kann den Module Manager nutzen?

Nur **ETW-Entwickler** (Entwickler-Konten) können den Module Manager sinnvoll nutzen. Normale Nutzer und Käufer brauchen ihn nicht — sie kaufen und installieren Module direkt.

## Technische Details

Der Module Manager läuft lokal auf deinem Computer. Er verbindet sich für bestimmte Aufgaben mit dem AAIA-Marketplace-Server, aber der Großteil der Arbeit findet auf deinem Rechner statt.
