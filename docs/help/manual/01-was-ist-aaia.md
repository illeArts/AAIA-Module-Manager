# Was ist AAIA?

## Kurz erklärt

AAIA steht für **AI-Assisted Integration Architecture** — zu Deutsch: KI-gestützte Integrationsarchitektur.

Einfach gesagt: AAIA ist ein System, das Entwickler dabei unterstützt, Erweiterungen (Module) für den **AAIAS-Server** zu erstellen, zu prüfen und zu veröffentlichen. Dabei hilft künstliche Intelligenz an bestimmten Stellen — zum Beispiel bei der Analyse von Fehlern oder beim Erstellen von Dokumentation.

## Wofür brauche ich AAIA?

Wenn du ein Modul für AAIAS entwickeln möchtest, brauchst du AAIA. Es ist das Werkzeug, das alle Schritte vom Anfang bis zur Veröffentlichung begleitet:

1. Du gibst eine Idee für ein Modul ein.
2. AAIA hilft beim Erstellen der Projektstruktur.
3. AAIA prüft, ob alles korrekt aufgebaut ist.
4. AAIA kompiliert den Code.
5. AAIA verpackt das Modul in eine `.aaiaext`-Datei.
6. AAIA signiert das Paket mit deiner Entwickleridentität.
7. AAIA lädt das Paket auf den Marketplace hoch.

## Was ist AAIAS?

**AAIAS** (AAIA Server) ist der Server, auf dem Module installiert und ausgeführt werden. Nutzer kaufen Module auf dem Marketplace und installieren sie auf ihrem eigenen AAIAS-Server.

## Was ist der Unterschied zu normaler Software?

AAIA-Module werden geprüft, signiert und verifiziert, bevor sie auf dem Marketplace erscheinen. Das schützt Nutzer davor, schädliche oder manipulierte Software herunterzuladen.

## Technische Details

AAIA Module Manager ist eine Windows-Desktop-Anwendung, die mit Avalonia und C# entwickelt wurde. Es benötigt das .NET 8 SDK zur Kompilierung.
